EQCat

Development record of this application.
brief summary:
A simple chatbot UI framework for an AI application. With a image of chat history, the application provides proper reply messages to their bosses or colleges for the User, including a domain model for segmenting and extracting text from WeChat chat history.
With the help of backend programs, EQCat follows a structure using different methods to ask the User to provide more information about the situation and problems they face, and reply tem the potential messages they could use.
